Quadratic planar spiral explorer

System: linear + quadratic terms, no constant term. The origin is always an equilibrium. Drag coefficients to change the local spiral geometry and finite-radius angular speed.

ODE

ẋ = a₁₁x + a₁₂y + b₂₀x² + b₁₁xy + b₀₂y²
ẏ = a₂₁x + a₂₂y + c₂₀x² + c₁₁xy + c₀₂y²
λ± = α ± iω, α = tr(A)/2, ω = √(det(A) − α²). For eigenvector v = p + iq, P = [p q] and Elin = σmax(P)/σmin(P) = √(λmax(PᵀP)/λmin(PᵀP)); undefined if ω = 0.
